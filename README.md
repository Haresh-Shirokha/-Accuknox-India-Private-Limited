# Social Network API

## Installation

1. Clone the repository:
   ```bash
   git clone <repo_url>
   cd social_network

   API Endpoints
POST /api/signup/: User signup
POST /api/login/: User login
GET /api/friend-requests/: List friend requests
POST /api/friend-requests/: Send friend request

### Final Steps

1. **Run Migrations**: After building the Docker containers, run the migrations to set up the database.
2. **Test the API**: Use Postman or any other API testing tool to interact with the API endpoints.
